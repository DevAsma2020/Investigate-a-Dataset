Titanic Data Analysis

Introduction:¶

The aim of this project is to analysis sample of Titanic data set and find out if the factor "being a mother with children onboard" played favor for them, in increasing number of survives among them(mothers).

Also, through this project, another question will be answered which is does the Class factor helped those who were in upper class to survive more than those who were in lowest class in Titanic.
